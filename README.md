# texty: convert your handwritten notes into text! (v1)

Check it out live at: https://textyocr.streamlit.app/<br>
A tool that converts uploaded images of handwritten notes into typed text for copy-pasting, storage, or simply figuring out what was actually written.<br>
Currently using TrOCR - plans to update to DeepSeek OCR (with stronger capabilities) soon!<br>
